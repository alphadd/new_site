RESPONSIVE WEBSITE

379 Media Works is a hybrid team of artists based in Italy. Their name comes from the CTRL+Z keyboard shortcut, which in computer media cancels the latest action performed.

Their need was an interactive portfolio showcase and an innovative website.

The result is an inventive website that offers no definitive layout. The user can drag & drop the photos as he wishes creating custom individual previews. Also each product page of the website can be altered by the administrator as he can change the layout from the CMS. The site is a responsive website with a fluid layout that adapts to all screen resolutions, ranging from a desktop computer to a mobile phone.
